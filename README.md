# Foundry Prompt Testing

A Python project for translating text into multiple languages, fully Dockerized for easy deployment and testing.

## Features

- Translate text into **French, Spanish, German, Italian, Arabic, Japanese, and Portuguese**.
- Docker support for consistent, isolated environments.
- Version-controlled with Git; `.env` and cache files are ignored.

## Requirements

- Python 3.11
- Docker
- Required Python packages (listed in `requirements.txt`)
